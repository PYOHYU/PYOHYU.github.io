### 1.3.1 명령 창의 출력 표시 형식 설정

```
format (style)
```

출력 형식을 설정. long 타입의 경우 소수점 아래 15자리까지 표현된다.

```Matlab
format long
pi
```

> 결과 : ans = 3.141592653589793

초기 출력 형식은 short.

```Matlab
format short
pi
```

> 결과 : ans = 3.1416
